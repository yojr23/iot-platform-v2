# Guia de Referencias ICONTEC para Documentacion Cientifica

Esta guia resume formatos operativos para reportes tecnicos y articulo cientifico.
Base normativa: `NTC 5613` y `NTC 4490`.

## 1. Libro

Formato:

`APELLIDO(S), Nombre(s). Titulo del libro. Edicion (si aplica). Ciudad: Editorial, ano.`

## 2. Articulo de revista cientifica

Formato:

`APELLIDO(S), Nombre(s). Titulo del articulo. En: Nombre de la revista. Vol. x, no. y (mes, ano); p. xx-yy. DOI (si aplica).`

## 3. Norma tecnica (ISO, IEC, IEEE, NTC)

Formato:

`ENTIDAD. Codigo de norma: titulo. Edicion. Ciudad: entidad editorial, ano.`

Ejemplos:

`INTERNATIONAL ORGANIZATION FOR STANDARDIZATION (ISO). ISO 9001:2015. Quality management systems - Requirements. Geneva: ISO, 2015.`

`INTERNATIONAL ORGANIZATION FOR STANDARDIZATION (ISO); INTERNATIONAL ELECTROTECHNICAL COMMISSION (IEC). ISO/IEC 27001:2022. Information security, cybersecurity and privacy protection - Information security management systems - Requirements. Geneva: ISO, 2022.`

## 4. Recurso web institucional

Formato:

`ENTIDAD. Titulo [en linea]. Ciudad: entidad responsable, ano (si aplica) [consultado: DD mes AAAA]. Disponible en: URL`

## 5. Buenas practicas obligatorias en este repositorio

1. No dejar enlaces sueltos sin referencia estructurada.
2. Citar fuentes primarias de estandares y especificaciones.
3. Registrar fecha de consulta para contenido web.
4. Evitar bibliografia no usada en el contenido.
5. Mantener orden alfabetico por entidad o autor.
